# 配置说明

## 基础配置

- `方案名称`
- `基础命名前缀`
- `导出目录`

界面里只输入一个基础前缀，实际输出文件会自动区分为：
- `*_CC_*`：恒流段
- `*_EIS_*`：阻抗段
- `*_ACT_*`：预处理段
- `*_REL_*`：脉冲前弛豫段
- `*_PULSE_*`：脉冲段

## 电池参数与 1C 依据

这里只保留和电流换算直接相关的量：
- `活性物质质量 (mg)`
- `理论比容量 (mAh/g)`

1C 计算逻辑：

```text
活性物质质量(mg) / 1000 = 活性物质质量(g)
活性物质质量(g) × 理论比容量(mAh/g) = 1C 对应容量(mAh)
1C 对应容量(mAh) / 1000 = 1C 对应电流(A)
```

例如：

```text
1 mg × 865 mAh/g = 0.865 mAh
0.865 mAh / 1000 = 0.000865 A
```

## 放电策略

通用字段：
- `电压上限`
- `电压下限`
- `放电前等待时间`
- `采样间隔`

采样间隔支持预设：
- `0`
- `0.001`
- `0.1`
- `1`

也支持直接输入自定义值。

## 放电电流

支持两种方式：
- 直接输入电流 `A`
- 输入倍率 `C`，由系统根据 1C 自动换算

## 电压取点

支持：
- `智能预设`
- `自定义列表`

智能预设输入：
- `初始 OCV`
- `平台电压`
- `截止电压`
- `目标点数`

界面会把最终分配出的电压点规整列出来，便于直接检查。

## 时间取点

支持：
- `智能预设`
- `自定义列表`

智能预设输入：
- `总时间`
- `前期持续时间`
- `平台持续时间`
- `后期持续时间`
- `目标点数`

界面会把最终分配出的时间点规整列出来，便于直接检查。

## 分阶段均匀时间点

适合已经知道前期、平台期、后期各自持续范围时使用。

每行格式：

```text
起点(min), 终点(min), 点数
```

## 预处理 + 正式测试

适合需要先做几步预处理，再进入正式原位阻抗测试的场景。

预处理每行格式：

```text
时长(s), 电流(A) 或 倍率(C)
```

## 脉冲放电取点

支持：
- 弛豫方式：`静置 / 恒流`
- 弛豫电流：`倍率模式 / 直接输入`
- 脉冲电流：`倍率模式 / 直接输入`
- `脉冲次数`
- `采样间隔` 预设和自定义

默认采样间隔是 `0.001 s`。

## IMP 参数

- `Use Open Circuit E as Init E`
- `Init E`
- `fh`
- `fl`
- `amp`
- `qt`

启用 `Use Open Circuit E as Init E` 后，`Init E` 会灰显。
