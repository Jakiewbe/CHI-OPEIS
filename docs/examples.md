# 示例场景

本文档给出 OPEISMaster 计划优先支持的三个典型场景。示例同时面向实验人员和开发者：

- 实验人员可以快速判断该选哪种模板。
- 开发者可以据此设计表单、校验逻辑和输出用例。

## 示例一：固定电压点原位阻抗

### 适用场景

- 需要比较不同电压状态下的阻抗变化。
- 更关注电位点与阻抗的对应关系，而不是严格的时间轴一致性。

### 建议模板

- 放电段：`CP`
- 阻抗段：`IMP`

### 典型输入

- OCV 测一次阻抗
- 放电电流：按样品容量换算
- 目标电压点：`2.7, 2.6, 2.5, 2.4, 2.3, 2.2, 2.1, 2.0, 1.5`
- 阻抗参数：`fh=100000`、`fl=0.01`、`amp=0.005`、`qt=2`

### 应提示的 Warning

- OCV 附近电压点可能被跳过。
- 电压点过密时，总实验时间会显著增加。
- `tech=cp` 切到 `tech=imp` 后，关键参数必须重写。

### 输出片段示例

```text
tech=cp
ic=0.000040136
ia=0
eh=3
el=2.7
tc=100000
ta=0.005
si=1
cl=1
prioe
run
save=D27

tech=imp
ei=2.7
fh=100000
fl=0.01
amp=0.005
qt=2
impautosens
impft
run
save=E27
```

## 示例二：固定时间点原位阻抗

### 适用场景

- 已经有基准放电曲线。
- 希望在累计放电时间的关键节点插入 EIS。
- 需要避免把 EIS 测试时间计入累计放电时间。

### 建议模板

- 放电段：`ISTEP`
- 阻抗段：`IMP`

### 典型输入

- OCV 阻抗：开启
- 放电电流：`0.0000865 A`
- 时间点：`2, 6, 10, 14, 18, 30, 50, 70, 90, 110, 130, 150, 170, 190, 210, 230, 250, 280, 310, 340, 370, 400, 425, 450 min`
- `eh=3`
- `el=1.5`
- `si=1`
- `fh=100000`
- `fl=0.01`
- `amp=0.005`
- `qt=2`

### 应提示的 Warning

- `fl=0.01` 且点位达到 24 个时，总实验时长会很长。
- 该方案输出的是带 EIS 中断的间歇放电曲线。
- 若命名规则不唯一，`fileoverride` 会带来静默覆盖风险。

### 输出片段示例

```text
tech=imp
eio
fh=100000
fl=0.01
amp=0.005
qt=2
impautosens
impft
run
save=OCVEIA

tech=istep
is1=0.0000865
st1=120
eh=3
el=1.5
si=1
cl=1
run
save=D002M

tech=imp
eio
fh=100000
fl=0.01
amp=0.005
qt=2
impautosens
impft
run
save=E002M
```

## 示例三：脉冲前后测阻抗

### 适用场景

- 需要比较脉冲前后阻抗变化。
- 关注高倍率或瞬态负载对界面状态的影响。

### 建议模板

- 步骤控制：`ISTEP`
- 阻抗段：`IMP`

不建议单纯依赖 `CP` 的时长切换去拟合单脉冲流程，因为它更容易出现“切换逻辑正确但行为不直观”的问题。

### 典型输入

- OCV 阻抗
- 基线放电段
- 脉冲前阻抗
- 高倍率脉冲
- 脉冲后阻抗
- 按循环次数重复

### 应提示的 Warning

- 高倍率脉冲可能导致瞬时极化明显，脉冲后阻抗不应直接与静态 OCV 阻抗等同解释。
- 如果脉冲次数较多，文件命名必须包含循环编号。

### 输出片段示例

```text
tech=istep
is1=0.0000865
st1=60
eh=3
el=1.5
si=1
cl=1
run
save=BASE01

tech=imp
eio
fh=100000
fl=0.1
amp=0.005
qt=0
impautosens
impft
run
save=PRE01

tech=istep
is1=0.00865
st1=1
eh=3
el=1.5
si=0.1
cl=1
run
save=PULSE01

tech=imp
eio
fh=100000
fl=0.1
amp=0.005
qt=0
impautosens
impft
run
save=POST01
```

## 示例对比

| 场景 | 推荐组合 | 优势 | 主要风险 |
| --- | --- | --- | --- |
| 固定电压点 | `CP + IMP` | 便于比较电位与阻抗 | OCV 邻近点易跳过 |
| 固定时间点 | `ISTEP + IMP` | 时间轴最清晰，适合自动化 | 放电曲线被 EIS 中断 |
| 脉冲前后 | `ISTEP + IMP` | 更适合固定电流、固定时长流程 | 瞬时极化与命名复杂度更高 |

## 给开发者的实现提示

如果你要把这些示例做成 GUI 模板，建议界面至少包含下面三个区域：

- 左侧参数区：按“样品信息 / 放电参数 / EIS 参数 / 输出命名”分组。
- 右上脚本预览区：实时展示纯命令文本。
- 右下验证日志区：集中展示 Warning、阻断错误和命名检查结果。

对于实验人员，选择模板时可以优先按这个原则判断：

- 想看不同电压状态，用固定电压点。
- 想按放电进程均匀追踪，用固定时间点。
- 想看脉冲扰动前后差异，用脉冲前后模板。
