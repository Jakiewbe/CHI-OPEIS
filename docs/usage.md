# 使用说明

## 启动

```powershell
.\.venv\Scripts\python -m chi_generator.main
```

或兼容入口：

```powershell
.\.venv\Scripts\python -m opeis_master.main
```

## 基本流程

1. 选择脚本类型。
2. 填写方案名称、基础命名前缀、导出目录。
3. 填写活性物质质量和理论比容量，系统会自动换算 1C。
4. 选择放电电流输入方式：倍率或直接电流。
5. 配置电压窗口、放电前等待时间和采样间隔。
6. 选择电压取点、时间取点、分阶段时间点、预处理或脉冲相关策略。
7. 调整 IMP 参数。
8. 在右侧检查摘要、点位分配、脚本预览和告警。
9. 没有 `Error` 时复制注释版或极简版脚本。

## 界面说明

- 左侧：输入区
- 右侧上方：告警和错误
- 右侧下方：注释版 / 极简版脚本预览

## 默认脚本行为

- 头部自动写入 `fileoverride / folder / cellon`
- 首段默认先测一次 OCV 阻抗
- 结束自动写入 `celloff / end`
- 文件名自动区分 `CC` 与 `EIS`

## 结果判定

- `Error`：阻止生成
- `Warning`：允许生成，但需要人工确认

## 典型告警

- EIS 点较多且 `fl = 0.01`
- OCV 附近取点偏密
- IMP 插入次数过多导致放电曲线间歇化
